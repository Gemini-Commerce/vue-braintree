# Introduction

VueBraintree plugin is plugin for Vue.js to quickly integrate the Braintree Drop In in a Vue.js application.

  • Works with: Credit Cards, Paypal, Apple Pay, Google Pay, etc.
  • Choose your own language. Available in English, French, Spanish, etc.
  • Fully customizable

Get started or play with the demo (see README.md to run them).
